# Onboarding Flow

A Next.js project that shows how to build a personalised onboarding form with [Formity](https://formity.app).

The form adapts its questions based on the user's answers — learners who pick frontend development see a different set of technologies than those who pick backend or mobile.

## What you'll learn

You'll see how to use the `switch` element to branch the flow based on a previous answer, and how `variables` lets you safely reference fields that only exist in certain branches when collecting the final result.

## Getting started

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
